# Democracy Now

## Directories
1. Model code goes into src/main/java/agile18/demo/model
2. Web interface controller code goes into src/main/java/agile18/demo/web
3. Static HTML/CSS/JS go into src/main/resources/static
4. Dynamic HTML templates go into src/main/resources/templates

## Steps to run
1. Have vs code
2. Get `Extension Pack for Java`
3. Get `Spring Boot Extension Pack`
4. Click run button at the top right
